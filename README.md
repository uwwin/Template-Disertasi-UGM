# Template-Disertasi-UGM
Template Disertasi S3 Departemen Ilmu Komputer dan Elektronika FMIPA UGM

# Cara Menggunakan
[] Download File
[] Modifikasi file .tex disesuaikan konten
[] Kompilasi dengan menjalankan ./gcompil.sh 
 (Linux Based) jika ingin menggunakan Windows maka buat file .bat sesuaikan perintah file .bat dengan isi file  ./gcompil.sh 
[] Cek File hasil yaitu disertasi.pdf
